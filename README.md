# Logistic-Regression-task
# 🧠 Breast Cancer Classification using Logistic Regression

## 📌 Project Overview
This project is a machine learning model that predicts whether a tumor is malignant (cancerous) or benign (non-cancerous) using Logistic Regression.

---

## 📊 Dataset
- Breast Cancer Dataset (CSV file)
- Features include tumor measurements like radius, texture, area, etc.
- Target variable:
  - M = Malignant (0)
  - B = Benign (1)

---

## ⚙️ Steps Performed
1. Data Loading using Pandas
2. Data Cleaning (removing unnecessary columns)
3. Converting categorical values to numeric
4. Splitting dataset into training and testing sets
5. Training Logistic Regression model
6. Making predictions
7. Evaluating model performance

---

## 🤖 Model Used
- Logistic Regression (from Scikit-learn)

---

## 📈 Results
- Accuracy: ~95%–98%
- Model performs well in distinguishing between malignant and benign tumors.

---

## 📊 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report (Precision, Recall, F1-score)

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
